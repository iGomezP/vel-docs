# RFC-0001 — Project Management Platform Evaluation

## Status

Draft

---

## Objective

Evaluate open-source project management platforms suitable for Vakzor Enterprise Lab (VEL).

The selected platform will become the official Application Lifecycle Management (ALM) system used to manage engineering work, documentation, architecture, operations, and software delivery.

---

## Why this RFC exists

VEL aims to emulate a real engineering organization.

Tool adoption must never be based on popularity or aesthetics.

Instead, every platform should be evaluated using objective engineering criteria.

---

## Candidate Platforms

- Plane
- OpenProject
- Taiga

Additional candidates may be added during the evaluation.

| Criteria | Weight | Notes |

| Scrum Support | 8 | Sprint management |

| Roadmap | 7 | Planning |
